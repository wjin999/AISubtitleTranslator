AI Subtitle Translator是一个基于 Python 的高性能字幕翻译工具，利用大语言模型（如 DeepSeek、OpenAI）提供高质量的 SRT 翻译。
它采用 asyncio 异步并发架构，能够显著提升长视频字幕的处理速度。
程序内置了 spaCy 智能合并功能，自动修复被时间轴切碎的短句，解决翻译语意不连贯的问题。
在翻译前，系统会生成全文剧情摘要，让 AI 充分理解故事背景。
翻译时采用滑动窗口机制携带上下文，确保每一句翻译都符合语境。
支持自定义术语表（Glossary），通过动态注入技术保证专有名词翻译的一致性。
内置的后处理模块会自动清洗多余的符号和不规范标点。
用户可以灵活配置 API 密钥、并发数量以及使用的模型。
默认针对 DeepSeek API 优化，同时也兼容 GPT-4 等其他 OpenAI 格式接口。
只需简单的一行命令行操作即可完成从读取到生成的全流程。
